### 2. Basic HTML Form
A simple form that sends data to an email address using the `mailto:` attribute.

**Features:**
- Form created with `<form>` tag  
- Uses `method="post"` and `action="mailto:..."`  
- Includes `<input>` and `<label>` elements for user entry  

**File:** `form.htm`
